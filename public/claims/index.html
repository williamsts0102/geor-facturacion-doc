Documentación de API: Libro de Reclamaciones
Registrar un Reclamo
Descripción

Esta API permite registrar un reclamo en el Libro de Reclamaciones de la tienda en línea de Geor Ecommerce.

Detalles de la API

Para registrar un reclamo, realice una solicitud POST a la siguiente URL:

  • URL: https://wts.georfacturacion.test/api/v1/ecommerce/claims/register
  • Método: POST
  • Autenticación: Se requiere autenticación para acceder a esta API. Asegúrese de proporcionar las credenciales de autenticación adecuadas en el encabezado de la solicitud.
Parámetros del Cuerpo de la Solicitud

El cuerpo de la solicitud debe contener la siguiente estructura JSON con todos los campos requeridos: 

      
{
  "full_name": "prueba",
  "document_type": "prueba",
  "document_number": "prueba",
  "email": "prueba",
  "phone": "prueba",
  "address": "prueba",
  "department": "prueba",
  "province": "prueba",
  "district": "prueba",
  "description": "prueba"
}

Todos los campos son requeridos para registrar un reclamo en el Libro de Reclamaciones.

Respuesta

La respuesta de la API será un objeto JSON que indica si el reclamo se guardó exitosamente o no: 

      
{
  "success": true,
  "message": "Reclamo guardado exitosamente."
}

El campo "success" indicará si el registro fue exitoso (true) o no (false). El campo "message" proporcionará un mensaje descriptivo sobre el resultado de la operación.
