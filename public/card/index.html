Registro de tarjeta

Descripción:Endpoint para registrar la tarjeta de usuario

URL:http://pasion.georfacturacion.test/api/v1/basic/card/register

Método:POST

Request:

    {
  "number": "4578412563541874596354925",
  "titular": "michael",
  "expiration_date": "10/23"
}

Respuesta

        {
  "data": {
    "id": 10,
    "number": "4578412563541874596354925",
    "titular": "michael",
    "expiration_date": "10/23"
  },
  "success": true,
  "message": "Tipo de documento creado de forma correcta."
}
Actualizar tarjeta

Descripción:Endpoint actualizar la tarjeta de usuario

URL:http://pasion.georfacturacion.test/api/v1/basic/card/update/[Identificador]

Método:PUT

Request:

    {
  "number": "4578412563541874596354925",
  "titular": "michael",
  "expiration_date": "20/10"
}

Respuesta

		{
  "data": {
    "id": 10,
    "number": "4578412563541874596354925",
    "titular": "michael",
    "expiration_date": "20/10"
  },
  "success": true,
  "message": "El tipo de documento seleccionado fue actualizado de forma correcta."
}
