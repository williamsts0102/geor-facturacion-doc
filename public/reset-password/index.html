Recuperar contraseña

Descripción: Sección donde se muestran los endpoints necesarios para que el usuario recupere su contraseña


Enviar código de verificación al correo electrónico del usuario

URL: http://pasion.georfacturacion.test/api/v1/ecommerce/reset-password/send-code

Método: POST

Request:

		{
  "email": "demo@gmail.com"
}

Respuesta

		{
  "success": true,
  "message": "Se ha enviado un código de verificación al correo electrónico ingresado, reviselo y siga las instrucciones."
}


Reenviar código de verificación al correo electrónico del usuario

URL: http://pasion.georfacturacion.test/api/v1/ecommerce/reset-password/resend-code

Método: POST

Request:

		{
  "email": "demo@gmail.com"
}

Respuesta

		{
  "success": true,
  "message": "Se ha enviado un código de verificación al correo electrónico ingresado, reviselo y siga las instrucciones."
}


Validar código de verificación

URL: http://pasion.georfacturacion.test/api/v1/ecommerce/reset-password/validate-code

Método: POST

Request:

		{
  "email": "demo@gmail.com",
  "code": "102030"
}

Respuesta

		{
  "success": true,
  "message": "El código fue validado de forma correcta."
}


Cambiar la contraseña

URL: http://pasion.georfacturacion.test/api/v1/ecommerce/reset-password/change-password

Método: POST

Request:

		{
  "email": "demo@gmail.com",
  "code": "622391",
  "password": "Demo#123",
  "password_confirmation": "Demo#123"
}

Respuesta

		{
  "success": true,
  "message": "La contraseña ha sido restablecida de forma correcta."
}
