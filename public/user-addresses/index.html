CRUD de direcciones de usuario

Se requiere autenticación y validacion para acceder a estas API. Asegúrese de proporcionar las credenciales adecuadas.

Registrar la dirección

Descripción: Endpoint para registrar la dirección del usuario

URL: http://pasion.georfacturacion.test/api/v1/ecommerce/addresses/register

Método: POST

Request: 

    {
  "description": "Tienda",
  "country_id": "PE",
  "address": "sjl. campoy",
  "extra": [
    {
      "national_division": "Departamento",
      "value": "Lima",
      "ubigeo": "10"
    },
    {
      "national_division": "Provincia",
      "value": "Barranca",
      "ubigeo": "1001"
    },
    {
      "national_division": "Distrito",
      "value": "Pativilca",
      "ubigeo": "100101"
    }
  ],
  "postal_code": "14",
  "default": true
}

postal_codeCódigo postalStringMáximo 15 caracteresNo
